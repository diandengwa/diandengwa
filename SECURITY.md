# 安全政策

点灯蛙 (Diandengwa) 致力于保护用户（尤其是未成年人）的数据安全与隐私。

## 报告安全漏洞

如果您发现任何安全漏洞，请通过以下方式报告：

📧 **安全问题**：security@diandengwa.ai

**请勿**在 GitHub Issues 中公开讨论安全漏洞。

## 安全承诺

- 🔒 本项目不含任何用户数据收集功能
- 👶 专为未成年人教育场景设计，合规优先
- 🛡️ 遵循儿童在线隐私保护原则
- 🔐 不存储、不传输、不泄露任何个人信息

## 依赖项安全

我们会定期更新依赖项以修复已知漏洞。请确保使用最新版本。
