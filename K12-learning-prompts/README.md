# K12 学段提示词库

本目录包含按学段细分的苏格拉底式启发引导提示词。

## 📁 目录结构

| 文件 | 适用年级 | 核心特点 |
|------|----------|----------|
| `小学低年级_1-3年级_兴趣保护版.md` | 1-3年级 | 情绪安抚、具象比喻、保护好奇心 |
| `小学高年级_4-6年级_会学能力版.md` | 4-6年级 | 三层溯源法、会学能力、小升初衔接 |
| `初中_7-9年级_中考意识版.md` | 7-9年级 | 六步分析法、中考导向 |
| `高中_10-12年级_元认知版.md` | 10-12年级 | 七步分析法、元认知训练 |

## 🎯 设计原则

所有提示词遵循 **Socratic Rule（苏格拉底规则）**：

1. **接纳情绪** — 先共情，不评判
2. **诊断错因** — 找出思维卡点
3. **具象比喻** — 用生活实例解释抽象概念
4. **启发追问** — 把思考的接力棒还给孩子

**严禁**：直接输出答案、直接给出解题步骤

## 🏷️ 文件格式

每个提示词文件头部包含元数据标签：

```markdown
@DiandengwaLevel: #年级/科目/知识点
@DiandengwaTag: C类-方法缺失
@DiandengwaVersion: 1.0.0
```

## 🚀 使用方式

### 方法一：复制粘贴
直接复制提示词内容，粘贴到任意 AI 助手的「角色设定」或「系统提示词」中。

### 方法二：MCP 协议接入
使用支持 MCP 的工具，通过 `SKILL.md` 配置自动加载。
